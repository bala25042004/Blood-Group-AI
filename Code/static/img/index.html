Predicting Blood Group
for Healthier Lives

Non-Invasive Diagnostic Machine Learning

Our advanced system leverages machine learning to analyze various features from retinal fundus and sclera eye images. This innovative, non-invasive approach aims to identify and predict a person's blood group, offering a safe alternative to traditional blood tests.

By identifying patterns in vascular and scleral features, we provide crucial insights that contribute to more secure and innovative healthcare diagnostics.

Capabilities

Our Prediction Capabilities

DATA PREPROCESSING

We meticulously clean, transform, and prepare raw image data, ensuring optimal quality for robust machine learning model training.

FEATURE EXTRACTION

Our system extracts key features like vessel tortuosity and redness from fundus and sclera images using advanced algorithms.

BLOOD GROUP PREDICTION

Utilizing a trained MLP model, we combine the extracted features to provide a prediction of the blood group.

TRAINING & ANALYSIS

Generate accuracy graphs, confusion matrices, and precision/recall reports to evaluate and improve the model's performance.

Predictive Diagnostics

Our predictions offer a new way to get quick blood group information, which is useful in emergencies or for proactive health management.

  • Non-invasive and rapid results
  • Enhanced emergency response
  • User-friendly and accessible

Health Insights

Beyond just predicting the blood group, our system aids in comprehensive data analysis to understand the correlation between eye features and blood types.

  • Trend analysis of eye features
  • Data-driven diagnostic recommendations
  • Research-grade data analysis
