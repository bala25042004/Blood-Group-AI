How It Works

This system predicts your blood group using advanced AI analysis of retinal fundus and sclera images. Completely non-invasive, fast, and accurate.

AI Analysis Pipeline

Our intelligent system follows a sophisticated 8-step process

1

Image Acquisition

Upload high-quality retinal fundus and sclera images through the secure web interface.

2

Preprocessing

Images are automatically resized, normalized, and enhanced to remove noise and improve clarity.

3

Feature Extraction

Advanced algorithms extract 10 unique vascular and color-based features from the images.

4

Feature Fusion

Fundus and sclera features are combined into a comprehensive feature vector for analysis.

5

AI Model Analysis

Deep learning neural networks process the feature vector through multiple layers.

6

Classification

The model outputs a confidence score for each blood group type.

7

Prediction

The highest confidence blood group is selected as the final prediction.

8

Result Display

The predicted blood group is displayed with analysis metrics and extracted features.

Extracted Features

10 unique metrics used for accurate blood group classification

Vessel Density

Proportion of blood vessel pixels in the retinal image.

Vessel Tortuosity

Measures the curvature complexity of retinal blood vessels.

Optic Disc Diameter

Size of the optic nerve head in the fundus image.

Macula Brightness

Average pixel intensity in the macula region.

Retinal Ridge Count

Ridge patterns indicating vascular branching complexity.

Sclera Redness Index

Measures blood vessel visibility in the sclera.

Sclera Texture

Brightness variation indicating vessel prominence.

Pigmentation Index

Color balance influenced by oxygenation levels.

Demographic Data

Age and health metrics for personalized analysis.

Vascular Profile

Comprehensive blood vessel pattern analysis.
